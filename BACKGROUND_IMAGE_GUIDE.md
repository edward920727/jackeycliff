# 背景圖片設置指南

## 無版權圖片來源

以下是一些提供無版權圖片（可商用）的網站：

1. **Unsplash** (https://unsplash.com/)
   - 搜索關鍵詞：`spy`, `secret mission`, `covert operation`, `intelligence`, `military operation`
   - 所有圖片都是免費且可商用的

2. **Pexels** (https://www.pexels.com/)
   - 搜索關鍵詞：`spy`, `secret`, `mission`, `intelligence`, `covert`
   - 所有圖片都是免費且可商用的

3. **Pixabay** (https://pixabay.com/)
   - 搜索關鍵詞：`spy`, `secret mission`, `intelligence`, `military`
   - 所有圖片都是免費且可商用的

## 推薦的圖片主題

適合《機密代號》遊戲的背景圖片主題：
- 間諜/特工主題
- 軍事行動
- 機密文件
- 暗室/密室
- 城市夜景（間諜風格）
- 軍事基地
- 情報中心

## 設置步驟

1. 從上述網站下載一張適合的圖片
2. 將圖片重命名為 `background.jpg` 或 `background.png`
3. 將圖片放到 `public` 資料夾中
4. 如果使用不同的檔名，請更新 `app/page.tsx` 中的圖片路徑

## 圖片規格建議

- **尺寸**：建議 1920x1080 或更大
- **格式**：JPG 或 PNG
- **檔案大小**：建議小於 2MB（可優化後使用）

## 範例圖片URL（Unsplash）

以下是一些可以直接使用的 Unsplash 圖片 URL（無版權）：

- 間諜主題：`https://images.unsplash.com/photo-1557804506-669a67965ba0?w=1920`
- 機密行動：`https://images.unsplash.com/photo-1519681393784-d120267933ba?w=1920`
- 軍事主題：`https://images.unsplash.com/photo-1505142468610-359e7d316be0?w=1920`

**注意**：建議下載圖片到本地使用，而不是直接使用外部 URL，以確保穩定性和載入速度。
